Step 1 — Install the Scraper Bookmarklet
The bookmarklet runs inside your already-logged-in Populi session, so it can read everything on the page. Drag the button below to your browser's bookmarks bar.
1
Make sure your bookmarks bar is visible
Chrome: Cmd/Ctrl + Shift + B  ·  Safari: View → Show Favorites Bar  ·  Firefox: View → Toolbars → Bookmarks Toolbar
2
Drag this button to your bookmarks bar
Don't click it here — drag it up to the bar. It'll appear as "📚 Scrape Populi".

⬆ Drag this button up to your bookmarks bar ⬆

📚 Scrape Populi
3
Navigate to your Populi course page
Go to the Lessons/Syllabus tab of your course in Populi. Make sure you're on a page that shows syllabus content, books, or lesson materials.
4
Click "📚 Scrape Populi" in your bookmarks bar
The bookmarklet will scan the page and ask for the course code (e.g., BL331). It then downloads two things:
📦 BL331-materials.zip — all attached files (PDFs, docs, etc.) bundled in a folder
📋 populi-scrape-BL331-*.json — metadata (books, lessons, descriptions)
A progress overlay appears in the bottom-right corner while files download.
💡 Run it on multiple Populi pages (Lessons, Syllabus, Files, Books) — each run downloads that page's files into a ZIP. Import all JSON files below and upload all ZIPs into SyllabusForge.
